# 13-team-project-fe

Frontend starter using React + TypeScript + Vite with Feature-Sliced Design
(FSD).

## Requirements

- Node.js 18+ (or 20+)

## Getting Started

```bash
npm install
npm run dev
```

## Scripts

- `npm run dev`
- `npm run build`
- `npm run preview`
- `npm run lint`

## Structure

```
src/
  app/
  pages/
  widgets/
  features/
  entities/
  shared/
```

## Aliases

- `@/` maps to `src/`.
